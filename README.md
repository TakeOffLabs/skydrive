# Skydrive

Simple client library for Microsoft Skydrive API with OAuth2 authentication

## Installation

Add this line to your application's Gemfile:

    gem 'skydrive'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install skydrive
